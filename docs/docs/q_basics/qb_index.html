QGIS Basics

QGIS in the Classroom

When you sit down at your PC in the QGIS sections of the course, the latest version of QGIS will already be installed, configured and ready to go. All you need to do is open the Start menu and search for QGIS, click the icon and the program will load up.

How to open a QGIS project

  • Click Project>Open in the menu bar.

  • Navigate to the project file you want to open and click Open.

How to switch between layers

Check and un-check the box next to the required layer to show and hide it.

How to add themes

  • Hide/show the layers necessary for your Theme.

  • Click the Themes button on the Layers panel .

  • Select Add Theme and give it a name.

  • You can now view this combination of layers at any time by opening the Themes menu again and checking the name of your created Theme.

How to draw a line feature

  • Select the relevant layer you want to draw on in the Layers Panel.

  • Set the layer to be editable with the Toggle Editing button .

  • Next click the Add Line Feature tool .

  • Start drawing your feature, Right-click when you have finished and confirm any details related to the line.

How to edit a line feature

  • Select the relevant layer selected and editable, click the Vertex Tool icon .

  • By clicking each point along your line you can re-position it.

  • Additional points can be created by clicking the cross at the central point of each line.

  • Points can be deleted by clicking them and then pressing the Delete key.

How to draw a polygon

  • Select the relevant layer in the Layers Panel.

  • Set the layer to be editable with the Toggle Editing button .

  • Next click the Add Polygon Feature tool .

  • Start drawing your feature, Right-click when you have finished and confirm any details related to the polygon.

How to select a feature

  • Select the relevant layer selected in the Layers Panel.

  • Click the Select Feature(s) tool. You can now select any features by clicking them or dragging a box around them. This will turn them yellow when selected.

  • This can also be done Freehand or by drawing a Polygon .

  • Click the Deselect Features From All Layers tool to de-select all selected features.

Tip

With the Select Features tool active you can select multiple features by Ctrl-Clicking each one individually.

Saving your changes

  • All changes you make to a layer you have made editable needs to be saved by clicking the Save button on the menu bar.

  • While you have un-saved changes for a layer, the edit icon on the Layers Panel will appear in red . When you have saved it will turn yellow.

  • Any changes you make to the framework of the map, such as whether a layer is visible or what colours it uses, are saved as a Project file (.qgz).

  • To save the Project as a whole, click Project>Save As…

Caution

Saving the Project file will not save any changes made to individual layers such as new lines or polygons you have drawn. These changes must be saved by clicking Save on the menu bar.

## QGIS in the Classroom

When you sit down at your PC in the QGIS sections of the course, the latest version of QGIS will already be installed, configured and ready to go. All you need to do is open the Start menu and search for **QGIS**, click the icon and the program will load up.\

## How to open a QGIS project

-   Click **Project\>Open** in the menu bar.

-   Navigate to the project file you want to open and click **Open**.

![](/assets/qb_open_file.gif){fig-align="center"}

## How to switch between layers

Check and un-check the box next to the required layer to show and hide it.\
\
![](/assets/qb_hide_layer.gif)

## How to add themes

-   Hide/show the layers necessary for your Theme.

-   Click the **Themes** button on the **Layers** panel ![](https://raw.githubusercontent.com/qgis/QGIS/3b6759013df710d7094c6c6a6fa77389ceebe72a/images/themes/default/mActionShowAllLayers.svg) .

-   Select **Add Theme** and give it a name.

-   You can now view this combination of layers at any time by opening the Themes menu again and checking the name of your created Theme.

![](/assets/qb_create_theme.gif){fig-align="center"}

## How to draw a line feature

-   Select the relevant layer you want to draw on in the **Layers Panel**.

-   Set the layer to be editable with the **Toggle Editing** button ![](https://raw.githubusercontent.com/qgis/QGIS/master/images/themes/default/mActionToggleEditing.svg).

-   Next click the **Add Line Feature** tool ![](https://raw.githubusercontent.com/qgis/QGIS/master/images/themes/default/mActionCaptureLine.svg) .

-   Start drawing your feature, **Right-click** when you have finished and confirm any details related to the line.

![](/assets/qb_draw_line.gif){fig-align="center"}

## How to edit a line feature

-   Select the relevant layer selected and editable, click the **Vertex Tool** icon ![](https://raw.githubusercontent.com/qgis/QGIS/3b6759013df710d7094c6c6a6fa77389ceebe72a/images/themes/default/mActionVertexTool.svg).

-   By clicking each point along your line you can re-position it.

-   Additional points can be created by clicking the cross at the central point of each line.

-   Points can be deleted by clicking them and then pressing the **Delete** key.

![](/assets/qb_edit_line.gif){fig-align="center"}

## How to draw a polygon

-   Select the relevant layer in the **Layers Panel.**

-   Set the layer to be editable with the **Toggle Editing** button ![](https://raw.githubusercontent.com/qgis/QGIS/master/images/themes/default/mActionToggleEditing.svg).

-   Next click the **Add Polygon Feature** tool ![](https://raw.githubusercontent.com/qgis/QGIS/master/images/themes/default/mActionCapturePolygon.svg) .

-   Start drawing your feature, **Right-click** when you have finished and confirm any details related to the polygon.

![](/assets/qb_draw_polygon.gif){fig-align="center"}

## How to select a feature

-   Select the relevant layer selected in the **Layers Panel**.

-   Click the **Select Feature(s)** ![](https://raw.githubusercontent.com/qgis/QGIS/bbcaeb937a0890f707b210824f95d6ddd12f022d/images/themes/default/mActionSelectRectangle.svg) tool. You can now select any features by clicking them or dragging a box around them. This will turn them yellow when selected.

-   This can also be done **Freehand** ![](https://raw.githubusercontent.com/qgis/QGIS/3b6759013df710d7094c6c6a6fa77389ceebe72a/images/themes/default/mActionSelectFreehand.svg) or by drawing a **Polygon** ![](https://raw.githubusercontent.com/qgis/QGIS/bbcaeb937a0890f707b210824f95d6ddd12f022d/images/themes/default/mActionSelectPolygon.svg).

-   Click the **Deselect Features From All Layers** ![](https://raw.githubusercontent.com/qgis/QGIS/bbcaeb937a0890f707b210824f95d6ddd12f022d/images/themes/default/mActionDeselectAll.svg) tool to de-select all selected features.\

![](/assets/qb_feature_select.gif){fig-align="center"}

::: callout-tip
With the **Select Features** ![](https://raw.githubusercontent.com/qgis/QGIS/bbcaeb937a0890f707b210824f95d6ddd12f022d/images/themes/default/mActionSelectRectangle.svg) tool active you can select multiple features by **Ctrl-Clicking** each one individually.
:::

##  Saving your changes

-   All changes you make to a layer you have made editable needs to be saved by clicking the **Save** button ![](https://raw.githubusercontent.com/qgis/QGIS/4fed6eae1643ad71f132035176488cf0a595df29/images/themes/default/mActionSaveAllEdits.svg) on the menu bar.

-   While you have un-saved changes for a layer, the edit icon on the **Layers Panel** will appear in red ![](https://raw.githubusercontent.com/qgis/QGIS/aecd96f5b7caf491d92c946e738c5f3f063a1019/images/themes/default/mIconEditableEdits.svg). When you have saved it will turn yellow.

-   Any changes you make to the framework of the map, such as whether a layer is visible or what colours it uses, are saved as a Project file (**.qgz**).

-   To save the Project as a whole, click **Project\>Save As...**

::: callout-caution
Saving the Project file will not save any changes made to individual layers such as new lines or polygons you have drawn. These changes must be saved by clicking Save on the menu bar.
:::
